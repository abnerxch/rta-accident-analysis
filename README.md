# 🚗 Análisis de Accidentes Viales — Etiopía

## 📌 Descripción
Análisis exploratorio del dataset RTA (Road Traffic Accidents)
con 12,316 registros y 32 variables.

## 🔍 Hallazgos principales
- El domingo tiene 3× más probabilidad de accidente fatal que el lunes
- "No distancing" es la causa #1 con 2,263 casos
- Conductores con 5-10 años presentan la paradoja de exceso de confianza
- Chi-square test: p=0.0004 — patrón estadísticamente significativo

## 🛠️ Herramientas
Python · pandas · seaborn · matplotlib · scipy
